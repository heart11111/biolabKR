# 비오랩 홈페이지

Astro + Tailwind CSS로 제작한 비오랩 정적 기업 홈페이지입니다.

## 주요 명령

```bash
npm install
npm run dev
npm run check
npm run build
npm run preview
```

## 콘텐츠 수정

- 회사 정보: `src/data/site.ts`
- 제품 정보: `src/data/products.ts`
- 원료/기술: `src/data/ingredients.ts`
- 공지/언론: `src/data/news.ts`
- 건강정보: `src/data/articles.ts`
- FAQ: `src/data/faqs.ts`

## 배포

GitHub Pages 프로젝트 사이트 기준입니다.

- `site`: `https://heart11111.github.io`
- `base`: `/biolabKR`
- GitHub 저장소 Settings > Pages에서 Source를 GitHub Actions로 설정합니다.
